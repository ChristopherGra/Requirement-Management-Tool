 # Requirements Management Tool

**WIP**

This tool processes requirement documents from various formats (Excel, CSV, PDF) and standardizes them into a unified structure for software projects with a high number of requirements and different sources of requirement documents.

## Important Note About Sample Data

**All content in this repository is generic and fictional.** The requirement documents contain only basic software application requirements created for demonstration purposes.

## Features

- **PDF Requirements Extraction**: Parse structured PDF requirements documents
- **Excel/CSV Processing**: Handle requirement imports from spreadsheet formats  
- **Column Mapping**: Interactive column mapping for different input formats
- **Standardization**: Convert all data to 16-column standard format
- **DOORS Integration**: Generate templates compatible with DOORS import
